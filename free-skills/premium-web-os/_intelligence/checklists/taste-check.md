# Taste Check (Gate 2)

> Score the static composition against the [`../taste.md`](../taste.md) rubric before motion/3D is added. **1 = template/sloppy · 3 = decent but generic · 5 = premium/cinematic/ownable.** Flagship target: ≥4 every axis, 5 on the focal axis. Below target → iterate; do not advance.

## Rubric (score each 1–5)

- [ ] **Composition** — deliberate, asymmetric, inevitable (not random blocks, not just a tidy grid). Score: ___
- [ ] **Hierarchy** — guides the eye like a film cut; clear primary → secondary. Score: ___
- [ ] **Typography** — editorial, confident, ownable; real scale + confident display size. Score: ___
- [ ] **Color / material** — controlled palette (~2 accents from tokens), atmospheric, material — not flat, not random gradients. Score: ___
- [ ] **Motion** — (intent only at this gate) choreographed/gravitational plan, not chaotic or absent. Score: ___
- [ ] **3D integration** — a brand metaphor with depth, or deliberately none — never decorative. Score: ___
- [ ] **Brand specificity** — unmistakably this brand, not "could be anyone". Score: ___
- [ ] **Originality** — fresh, ownable execution; not template/derivative. Score: ___
- [ ] **Polish** — obsessive detail, no "components bolted together" seams. Score: ___
- [ ] **Performance perception** — reads instant, smooth, weightless (not janky). Score: ___

## Focal axis

- [ ] Focal axis for this scene declared.
- [ ] Focal axis scores 5.
- [ ] All other axes score ≥4 (flagship). Below-flagship scenes hit their stated target.

## First-Viewport rule

- [ ] First viewport reads premium **before any scroll**.
- [ ] Contains a dominant claim.
- [ ] Contains clear spatial hierarchy.
- [ ] Contains one memorable visual object or scene.
- [ ] Uses intentional typography and a controlled palette.
- [ ] Zero placeholder aesthetic.

## Visual-Silence rule

- [ ] 40–60% of the composition is visually quiet — not every region is filled.
- [ ] The quiet space makes the focal point land.

## One focal point

- [ ] Exactly one visual focal point per scene; everything else is support.
- [ ] One dominant idea per viewport — the screen says one thing, not two.

## Anti-pattern sweep (any hit = not premium)

- [ ] No generic purple→blue/pink SaaS gradient.
- [ ] No center-aligned-everything (>60% centered text).
- [ ] No equal-card icon grid standing in for story.
- [ ] No emoji-as-icons or Unicode-glyph icons (✦ ◈ ⌥).
- [ ] No inconsistent radii/shadows or random font pairings.
- [ ] No generic "AI-powered solution" copy.
