# 01 — Reference Board

**Purpose:** Build a small, deconstructed reference board so art direction stands on principles, not copies.

## Inputs

- The intake summary from [`00-intake.md`](./00-intake.md).
- [`../reference-deconstruction.md`](../reference-deconstruction.md) — the deconstruction axes and the no-copy rule.
- [`../taste.md`](../taste.md) — premium vs cheap signals.
- [`../brand-worlds.md`](../brand-worlds.md) — the brand world this scene belongs to.

## Steps

1. **Gather references against a fixed quota** — enough to triangulate, not a moodboard dump:
   - 3 typography references,
   - 3 motion references,
   - 2 3D references,
   - 2 layout references,
   - 2 **negative** references (premium-adjacent work that fails — name *why* it reads cheap).
2. **Deconstruct each** along the axes in [`../reference-deconstruction.md`](../reference-deconstruction.md): first-viewport composition, type system, grid + where it breaks, color/material, hero metaphor, motion behavior. Capture the *principle*, never the asset.
3. **Tag each principle** as adopt / adapt / avoid for this brand. The negative refs feed the avoid list.
4. **Cross-check against the brand world.** Discard any principle that fights [`../brand-worlds.md`](../brand-worlds.md) — a borrowed principle that isn't on-brand is noise.
5. **Do not copy** a specific site, creator, or brand. If a principle can't be restated as a generalizable rule, it's a copy — drop it.

## Output / artifact

A **reference board**: each entry = source role (e.g. "typography ref 2"), the extracted principle, adopt/adapt/avoid, and the brand-fit note. Negative refs included.

## Gate

Every entry is a principle (not "make it look like X"), the avoid list is populated, and nothing on the adopt list contradicts the brand world.
