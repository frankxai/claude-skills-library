## Artistic Styles — The Visual Traditions

Beyond Gate alignment, each image can be rendered in distinct artistic traditions. These styles carry their own philosophy and aesthetic DNA.

### Classical Masters

```
┌─────────────────────────────────────────────────────────────────────────┐
│  DAVINCI                                                                 │
│  ────────                                                                │
│  Renaissance mastery, sfumato technique, anatomical precision            │
│  Chiaroscuro lighting, golden ratio composition, oil painting texture    │
│  Muted earth tones with divine light accents                            │
│                                                                         │
│  Best For: Guardian portraits, philosophical scrolls, anatomical studies │
│  Example:  aios viz guardian shinkami --style davinci                    │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│  BAROQUE                                                                 │

│  Deep shadows pierced by divine light rays                              │
│                                                                         │
│  Best For: Fire Gate scenes, dramatic moments, spiritual revelations     │
│  Example:  aios viz portrait "The Prophet" --style baroque --gate fire   │
└─────────────────────────────────────────────────────────────────────────┘
```
