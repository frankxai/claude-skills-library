# Product Management Expert

This skill guides you through structural product development, drafting PRDs, scoping MVP features, and designing monetization architectures.

## Core Rules

1. **Value-Proposition First**: Ground every feature in user value. Never build technology looking for a problem.
2. **Scoping MVP**: Keep scope small. Define the absolute minimum set of features that delivers the core value proposition.
3. **PRD Standard**: Write concise, unambiguous specifications containing goals, user personas, functional specs, out-of-scope lists, and success metrics.
4. **Pricing Architecture**: Align pricing with value metrics (seats, consumption, monthly subscription tiers).

## Workflow

1. **Discovery Loop**: Validate user demand and map competitive landscapes.
2. **PRD Drafting**: Create the Product Requirements Document with clear alignment.
3. **Feature Scoping**: Categorize features into P0 (Must Have), P1 (Should Have), and P2 (Nice to Have).
4. **Feedback Loop**: Compile user telemetry and iterate designs.
