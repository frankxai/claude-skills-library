# `_intelligence/` — the Premium Web OS canon

The Premium Intelligence Web OS is a control plane for creating cinematic, AI-native brand websites. It turns taste, design, motion, 3D, copy, and quality gates into explicit constraints so agents can execute with less ambiguity.

## Read order

**Always, before building:**
1. [`operating-system.md`](./operating-system.md) — how the OS works + the build sequence.
2. [`taste.md`](./taste.md) — what premium means, operationally + the 1–5 rubric.
3. [`brand-worlds.md`](./brand-worlds.md) — the brand world + its real token source (the token-binding contract).

**Scoped to the task:**
- [`design.md`](./design.md) — visual grammar, token categories, component standards, responsive doctrine.
- [`motion.md`](./motion.md) — motion grammar, easings, timing, GSAP/Motion rules.
- [`three-webgl.md`](./three-webgl.md) — 3D metaphors, per-scene spec, budgets, fallbacks.
- [`copywriting.md`](./copywriting.md) — voice, page copy architecture, CTA rules.
- [`reference-deconstruction.md`](./reference-deconstruction.md) — how to use references without copying.

**Process + governance:**
- [`agent-manual.md`](./agent-manual.md) — how an agent runs the OS end to end (incl. multi-agent mode).
- [`quality-gates.md`](./quality-gates.md) — the gates and when they fire.
- [`performance.md`](./performance.md) · [`accessibility.md`](./accessibility.md) · [`implementation-standards.md`](./implementation-standards.md)
- [`decision-log.md`](./decision-log.md) — durable record of decisions, dep additions, and TODOs.

## Folders

- [`workflows/`](./workflows/) — the 10-step pipeline (00-intake → 09-handover).
- [`prompts/`](./prompts/) — ready prompts for each role (page build, scene brief, motion director, visual QA, etc.).
- [`templates/`](./templates/) — fill-in artifacts (scene brief, page spec, component spec, motion spec, visual audit, handover).
- [`checklists/`](./checklists/) — preflight + per-discipline checks + the single release gate.
- [`examples/`](./examples/) — worked flagship page specs for Starlight, Arcanea, FrankX.

## The minimum bar

A page is premium when every visual decision supports status, clarity, trust, tension, memorability, and narrative control — and when it passes [`checklists/release-check.md`](./checklists/release-check.md). The difference between template and premium output is scored by the rubric in [`taste.md`](./taste.md).
