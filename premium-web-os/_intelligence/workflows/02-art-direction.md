# 02 — Art Direction

**Purpose:** Lock the dominant idea and the visual system — palette, type, material, motion intent, 3D metaphor — before any scene brief is written.

## Inputs

- The reference board from [`01-reference-board.md`](./01-reference-board.md).
- [`../taste.md`](../taste.md) — one-dominant-idea, premium signals, the rubric.
- [`../brand-worlds.md`](../brand-worlds.md) + the brand's real token source.
- [`../design.md`](../design.md) — token categories (color, type, space, components).

## Steps

1. **Lock the dominant idea** for the page in one sentence: what the first viewport says, and the single feeling it controls (status / clarity / trust / tension / memorability / narrative control). One idea per viewport — if it says two things it says nothing.
2. **Bind the palette** from the brand token source named in [`../brand-worlds.md`](../brand-worlds.md): a base + ≈2 accents. Reference token names, not raw hex. High contrast where it matters; controlled everywhere else.
3. **Set the type system:** display face + size + tracking, body face + scale, the one confident display moment. Authority comes from type before effects (taste principle 5).
4. **Define materiality:** glass / metal / paper / light / shadow / signal — chosen on purpose, consistent radii and shadows. No random surfaces.
5. **State the motion intent** (not the implementation): the single cinematic beat this page earns, and what stays still. Detail lands in [`06-motion-pass.md`](./06-motion-pass.md) per [`../motion.md`](../motion.md).
6. **Choose the 3D metaphor per scene** — a brand metaphor with depth, or *none*. No decorative blobs. If 3D, note the fallback obligation now.
7. **Reserve visual silence:** mark where the 40–60% quiet lives so the focal point can land.

## Output / artifact

An **art-direction lock**: dominant idea, palette (token names), type system, materials, motion intent, per-scene 3D metaphor (or "none"), quiet zones. This is the contract the scene briefs inherit.

## Gate

One dominant idea named; palette/type/material bound to real tokens (no invented values); motion intent is a single beat; every scene's 3D is a metaphor or explicitly none.
