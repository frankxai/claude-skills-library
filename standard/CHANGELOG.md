# Changelog

All notable changes to the Agent Skill Standard are recorded here. The format follows [Keep a Changelog](https://keepachangelog.com/), and the standard adheres to semantic versioning.

## [1.0.0] — 2026-06-22

### Added
- Initial specification ([`SPEC.md`](./SPEC.md)) on top of Anthropic Agent Skills.
- Six-level Skill Maturity Model ([`MATURITY-MODEL.md`](./MATURITY-MODEL.md)): L0 Prompt to L5 Composed.
- Risk-tier model (T0–T4) tying governance to possible harm.
- Conformance checklist ([`CONFORMANCE.md`](./CONFORMANCE.md)) and a zero-dependency validator ([`validate-skill.mjs`](./validate-skill.mjs)).
- Reference skills: `summarize-notes` (L1) and `customer-discovery-synthesis` (L4).
